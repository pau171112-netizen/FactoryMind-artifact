# Claude Code Notes

This is a Vite React app converted from a Claude artifact.

Guidelines:

- Work inside this repository only.
- Use `npm run dev:claude`; it serves the same canonical URL as Codex: `http://localhost:5173`.
- If port `5173` is busy, stop the existing dev server before starting a new one.
- Prefer focused edits in `src/App.tsx` unless adding a real new module is necessary.
- Run `npm run build` before handing off.
- Do not edit generated folders such as `node_modules` or `dist`.
- Use Git commits as handoff points between Claude Code and Codex.
- For artifact-style traceability, create or ask for a Git tag such as `artifact-v0.1.1` after a meaningful change.
